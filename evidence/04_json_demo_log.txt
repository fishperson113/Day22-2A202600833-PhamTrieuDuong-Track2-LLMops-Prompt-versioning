[Valid JSON] ✅ Pass
  Input:  {"name": "Alice", "age": 30}

  "name": "Alice",
  "age": 30
}

[Markdown fences] 🔧 JSON đã được sửa thành công
✅ Pass
  Input:  ```json
{"name": "Bob"}
```

  "name": "Bob"
}

[Single quotes] 🔧 JSON đã được sửa thành công
✅ Pass
  Input:  {'name': 'Charlie', 'score': 95}

  "name": "Charlie",

}
